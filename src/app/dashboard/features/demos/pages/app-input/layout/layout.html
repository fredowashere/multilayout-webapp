Flexgrid fundamentals

The utility class flexgrid lets you define a flexible grid from 1 to 5 columns, which is sort of ideal for forms.
The CSS variable --flexgrid-col-min-width lets you control the minimum width at which a flexgrid item wraps to the next line.
The CSS variable --flexgrid-gap lets you control the space between flexgrid items.

Apply the class flexgrid--no-grow if you don't want the elements to grow when then first elements has wrapped to the next line.
